# World — multi clock
Simple international multi clock on your tray.

## Download
<b>MacOS</b> — [DMG Installer](https://tarutin.github.io/world-clock/builds/World.dmg) (50MB)

## Contributing

```bash
$ git clone https://github.com/tarutin/world-clock.git
$ cd world-clock
$ npm install
$ npm start
```
.. and Package the app
``` bash
$ npm run build
```
